Initial Setup

Inventory Groups

Codes that contain costing information for inventory items.

Each group contains:

  • Group Code
  • Description
  • Cost Locations
  • Cost Codes by Level

Inventory Locations

Each entry represents a location where inventory can be stored, transfered to, or transfered from.

Each Location contains:

  • Code
  • Name
  • Default Location (Y/N)
  • F/A Department
  • Workshop (Y/N)
  • Parent Location

A location could be set up for each job site so tracking of job specific tools is easier. Another option is to do person specific but this can also be tracked by setting the responsible person when doing an Inventory Transaction.


Inventory Items

Each entry represents an item that is in inventory. Inventory items can be part of an inventory group and will inherit the groups properties. Inventory items can be transfered between inventory locations.

Each Item contains:

  • Code
  • Description
  • Group
  • Commodity Code

The Inventory Code must be a unique value for each line. This could be used to store Tool# or serial number. Could possibly be setup to auto assign so giving new tools an inventory number is automatic.


Maintaining Inventory

Adding New Items

To add new items to the inventory, they need to first have an inventory line created.

Add Single Item

  1. Go to "Inventory > Inventory Items" and click "add".
  2. Input the Inventory Code
  3. Fill in the Description
  4. Select the applicable Inventory Group

Returning to the "Inventory Items" screen should now show the new Inventory Item.

Add Multiple Items

  1. Go to "Inventory > Inventory Items"
  2. Click "BULK"
  3. Click "add"
  4. Open the downloaded excel file and click "Enable Editing"
  5. Columns A, C, and E must be filled out but other columns can be as well.
  6. Copy the entire workbook
  7. Paste back into the Coins screen
  8. Click "Save"
